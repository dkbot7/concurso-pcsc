Networks and Internet

A computer network is a group of connected devices that share information. The Internet is the largest global network.

ConceptMeaning
IP AddressUnique identifier for each device on the network (like a postal address)
IPv4Format: 4 numbers separated by dots (e.g., 192.168.0.1) — limited addresses
IPv6Longer format — created because IPv4 addresses were running out
TCP/IPCore protocol suite of the internet — ensures data delivery and addressing
HTTP/HTTPSProtocol for websites — HTTPS is the secure encrypted version
DNSTranslates domain names (google.com) into IP addresses — internet's phone book

3. MALWARE AND CYBER ATTACKS

ThreatHow it worksKey difference
VirusCopies itself into files; needs user to executeRequires human action to spread
WormSelf-replicates through networks automaticallyNo human action needed to spread
Trojan HorseDisguises itself as legitimate softwareDeceives user into installing it
RansomwareEncrypts files and demands ransom paymentExtortion — demands payment (often crypto)
SpywareRecords user activity secretlySurveillance — steals data silently
KeyloggerRecords every keystrokeTargets passwords and credentials
PhishingFake emails/sites to steal credentialsSocial engineering — not a virus
Key distinction: Phishing and social engineering exploit PEOPLE. Virus/worm/trojan exploit SOFTWARE vulnerabilities.

Internet of Things (IoT)

Everyday objects connected to the internet: security cameras, cars, smartwatches, locks. For police investigations, IoT devices are valuable sources of digital evidence.

Investigation note: A home camera, car GPS, or smartwatch may contain crucial evidence of a crime — time, location, or visual record.

PRACTICE QUESTIONS

Q1. A criminal encrypts all files on a company's server and demands payment in cryptocurrency to restore access. This attack is called:

  • A) Phishing
  • B) Ransomware
  • C) Worm
  • D) Spyware
Answer: B — Ransomware encrypts data and demands ransom. "Ransom" = payment for release.

Q2. A company uses software entirely through a browser — nothing installed on local machines. This cloud model is:

  • A) IaaS
  • B) PaaS
  • C) SaaS
  • D) DaaS
Answer: C — Software as a Service (SaaS) delivers applications through the browser. Examples: Gmail, Office 365.

Q3. The three pillars of information security are:

  • A) Control, Integrity, Access
  • B) Confidentiality, Integrity, Availability
  • C) Compliance, Infrastructure, Audit
  • D) Cryptography, Identification, Authentication
Answer: B — The CIA triangle: Confidentiality, Integrity, Availability.
