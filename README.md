## Como usar

### Imprimir as apostilas
1. Abra qualquer arquivo `.html` no navegador (Chrome ou Edge)
2. `Ctrl + P` → **Salvar como PDF** ou imprimir direto
3. Para 2 páginas por folha A4: nas configurações de impressão, selecione **2 páginas por folha**
4. Formato A5 já configurado automaticamente

### Ouvir os áudios
- Baixe os arquivos `.mp3` da pasta `AUDIOS_NARRADOS/` para o celular
- Ouça no trajeto, academia, ou antes de dormir
- Os arquivos `_NARRADO` são os roteiros narrativos (recomendados para primeira escuta)
- Os demais são a leitura completa do conteúdo (recomendados para revisão)

## Destaques do conteúdo

- Crimes hediondos — Lei 8.072/1990 com tabela completa
- Lei Maria da Penha — 5 tipos de violência, medidas protetivas
- Organizações criminosas — Lei 12.850/2013, meios especiais de investigação
- Abuso de autoridade — Lei 13.869/2019, dolo específico, condutas tipificadas
- LGPD na investigação policial
- Lavagem de dinheiro — 3 fases, COAF, sinais de alerta
- Sistema interamericano de direitos humanos — CIDH e Corte IDH
- Condenações do Brasil na Corte IDH
- Organização policial — PC, PM, PF, Polícia Penal, Guarda Municipal
- Prescrição disciplinar — prazos e procedimentos (sindicância x PAD)
